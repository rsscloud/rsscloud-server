# @rsscloud/core

Core primitives for [rssCloud](https://github.com/rsscloud/rsscloud-server) — subscriptions, notifications, and feed-update processing.

> **Status:** Early scaffolding. Logic is being migrated from `@rsscloud/server`.

## Install

```bash
pnpm add @rsscloud/core
```

## Usage

```ts
import { version } from '@rsscloud/core';

console.log(version);
```
